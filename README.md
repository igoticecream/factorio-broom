# Broom
A simple selection tool to sweep away unwanted elements from your Factorio world.

Select an area on your current surface and remove exactly what you want—decoratives, nuclear scorchmark, biter decals, rocks, trees, cliffs, and more. Keep your factory clean and organized with ease.

Customize cleanup for each player in the settings window that opens with the broom tool.

## Installation

[Download on the Mod Portal](https://mods.factorio.com/mod/broom)

## Usage
- Left mouse drag: Applies the actions selected in the Broom settings window. Available options:
    - Trees
    - Rocks
    - Cliffs
    - Resources
    - Atomic bomb scorchmarks
    - Biter creep
    - Decoratives (remove all, remove from artificial tiles only, or regenerate)
    - Corpses (biter corpses, explosion scorchmarks, tree stumps, building remnants)
    - Entities in editor mode (flamethrower ground flames, living enemies, and unit spawners)

## Bonus
Includes an option that prevents the aftermath of an atomic bomb (scorchmarks, nuclear tiles, etc.)
